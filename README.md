### Installation

1. Download .zip [__CLICK HERE__](https://github.com/franciscoblancojn/google-sheets-api-wordpress/archive/refs/heads/master.zip) in repository [https://github.com/franciscoblancojn/google-sheets-api-wordpress](https://github.com/franciscoblancojn/google-sheets-api-wordpress)
2. Go to admin of site wordpress, in __Plugins -> Add New__
3. Click in button __Upload Plugin__
4. Click in buttom __Select File__, and Upload file
5. Active Plugin
